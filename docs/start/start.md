# 快速上手

## 安装

::: code-group
```bash [pnpm]
# pnpm方式【推荐】
pnpm install lwu-css
```
```bash [npm]
# npm 方式
npm install lwu-css
```
```bash [cnpm]
# cnpm 方式
cnpm install lwu-css
```
```bash [yarn]
# yarn 方式
yarn add lwu-css
```
:::

## 使用

+ #### 入口文件引入
```ts
import 'lwu-css/dist/css/index.css'
```

+ #### 项目中愉快使用
```html
<div class="lwu-flex lwu-justify-center">
    hello lwu-css
</div>
```
